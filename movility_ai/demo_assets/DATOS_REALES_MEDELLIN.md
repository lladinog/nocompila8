## ⭐ Eco Score - Fórmula

```python
def calcular_eco_score(modo):
    scores = {
        "caminando": 100,
        "bicicleta": 100,
        "metro": 90,
        "metrocable": 90,
        "tranvia": 90,
        "bus": 70,
        "moto": 40,
        "carro": 20
    }
    return scores.get(modo.lower(), 50)
```

### Clasificación
- **90-100:** 🟢 EXCELENTE - Transporte sostenible
- **70-89:** 🟡 BUENO - Transporte eficiente
- **40-69:** 🟠 REGULAR - Considerar alternativas
- **0-39:** 🔴 BAJO - Alto impacto ambiental
